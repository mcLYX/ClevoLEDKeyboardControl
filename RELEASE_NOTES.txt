ClevoLEDKeyboardControl v1.4.0

新增功能：
- 新增“渐变循环”灯效：在用户设定的颜色序列中按顺序平滑过渡，支持 RGB 线性插值和 HSV 空间插值两种算法，可配置停留时长、过渡时长、过渡亮度百分比。
- 音乐模式新增“频率分布”响应模式：根据实时三频（低/中/高）能量比例映射颜色，不做节拍检测，颜色连续变化。可调整基础亮度、峰值亮度和色调偏移。
- 音乐模式新增“人耳等响度补偿（A-weighting）”选项：按 IEC 61672-1 标准补偿人耳等响曲线，避免低频能量压制中高频导致“几乎全是低频”。
- 音乐模式新增“动态范围”选项（不启用 / 仅峰值 / 峰值与下限）：动态跟踪峰值与底噪，适应不同响度音源和“响度战争”压缩音源，让灯光能反映细微响度差。
- 新增全局“亮度 Gamma 校正”：高级选项卡可调 0.8~3.0，影响所有灯效的亮度映射曲线，gamma>1 增加对比度。

修复和优化：
- 修复“仅亮度脉冲”模式颜色仍循环切换的问题：现在固定使用单个颜色，只有亮度随节拍脉冲，UI 也改为单色选择器。
- 修复频率分布模式开启等响度补偿后亮度上不去的问题：亮度改用原始能量，等响度补偿只影响颜色比例。
- 修复低音量时颜色在基色与频段比例色之间突然跳变（如绿→红）的问题：改用平滑混合过渡。
- 修复频率分布模式色调偏移在静音/低频时不生效的问题。
- 安装器新增 C:\Program Files (x86)\ControlCenter\DCHU 路径搜索，并支持从 assets 目录打包内置 DLL。

预设兼容：
- 新增字段（FreqHueOffset、EqualLoudness、DynamicRange、BrightnessGamma）均向后兼容，旧预设文件加载时使用默认值。

安装：
- 下载 ClevoLEDKeyboardControlSetup.exe。
- 以管理员身份运行安装器。
- 安装器会优先从 C:\Program Files (x86)\ControlCenter 复制 InsydeDCHU.dll，找不到时使用内置备用 DLL。

卸载：
- 在 Windows 设置 > 应用中卸载。
- 或再次运行安装器选择卸载。

说明：
- 支持 Windows 10/11 x64。
- 采用 GPL-3.0 协议。

ClevoLEDKeyboardControl v1.3.0

新增功能：
- 音乐模式加入自适应鼓点检测，响应更接近鼓点和节奏。
- 音乐内置预设调整为“通用”，默认使用更灵敏的节拍脉冲参数。
- 应用场景配置支持切换到音乐模式。
- 新增 Windows 通知闪烁，收到系统通知时可短促闪烁键盘。
- 音乐模式参数、通知闪烁、应用场景和亮度管理相关设置已纳入诊断和配置流程。

修复和优化：
- 优化音乐模式延迟和稳定性，减少响应滞后。
- 调整音乐预设参数，降低长时间保持低亮度的概率。
- 修复音乐模式基础亮度和全局亮度之间的冲突。
- 修复通知闪烁只在音乐模式下触发的问题。
- 修复保存设置后可能意外回到 RGB 循环的问题。
- 优化应用场景配置的颜色和亮度管理逻辑。

安装：
- 下载 ClevoLEDKeyboardControlSetup.exe。
- 以管理员身份运行安装器。
- 安装器会优先从 C:\Program Files (x86)\ControlCenter 复制 InsydeDCHU.dll，找不到时使用内置备用 DLL。

卸载：
- 在 Windows 设置 > 应用中卸载。
- 或再次运行安装器选择卸载。

说明：
- 支持 Windows 10/11 x64。
- 采用 GPL-3.0 协议。
